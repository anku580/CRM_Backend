Customers Endpoint

  • GET: Get All customers - http://localhost:3000/getAllCustomers
  • GET: Get customer by id - http://localhost:3000/getCustomer/{id}
  • POST: Create customer - http://localhost:3000/createCustomer
  • PUT: Update customer - http://localhost:3000/updateCustomer/{id}
  • DEL: Delete customers - http://localhost:3000/deleteCustomer/{id}

To test the remaining endpoints, please use POSTMAN or CURL
